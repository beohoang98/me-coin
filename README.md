# Demo
## Setup with docker 

- step 1 
```
docker-compose build
```

- step 2
```
docker-compose up
```

- Connection string

```
mongodb://admin:admin@localhost:27017
```

## Start

- setup
```
npm install
```

- node 1
```
npm run node_1
```

- node 2
```
npm run node_2
```
